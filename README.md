# Slow and Steady

A Socratic, bite-sized app-building methodology skill for AI coding agents like Claude Code, OpenCode, and Cursor.

## Overview

This tool provides a structured, thoughtful approach to application development, acting as an installable "skill" for compatible AI assistants. It helps manage complexity by guiding the agent through a step-by-step, methodical process.

## Installation / Usage

You can add this skill to your agent using `npx`:

```bash
npx skills add Entropy-rgb/slow-and-steady
```

This will install the skill directly from the repository.

## Supported Agents
- Claude Code
- OpenCode
- Cursor
- Antigravity (and other AI agents supporting npm-based skills)
